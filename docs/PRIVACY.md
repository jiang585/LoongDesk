# 御案隐私说明

御案不提供账户、云同步、广告、遥测或后台剪贴板监听。

## 只保存在本地

- 待办、关心库、来源设置、新闻元数据与缓存
- 聊天会话、AI 整理草稿、非敏感偏好
- AI 整理提案的应用、差异与撤销历史
- 窗口位置、显示器、DPI、托盘驻留与快捷键偏好
- DeepSeek API Key：使用 Stronghold 加密保险库单独保存

JSON 备份不包含保险库密码或 API Key。

## 会访问网络的操作

1. 用户向“小安子”发送问题时，问题、最近 20 轮对话及当前选择模块中最多 50 条/20,000 字上下文会发送给 DeepSeek。
2. 用户刷新奏报或粘贴单独的 HTTPS 链接时，Rust 网络层会访问该公开地址并提取标题、摘要和链接。
3. 用户主动打开新闻原文时，系统默认浏览器会访问新闻地址。
4. 用户在设置页主动检查更新时，更新器会访问本仓库 GitHub Release 的更新元数据；安装包必须通过 Tauri 更新签名校验。

本地导入的 TXT、Markdown、HTML 与 PDF 只在本机提取文字，不会上传。除此之外，御案不上传本地数据库，也不会后台读取系统剪贴板。
