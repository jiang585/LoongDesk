# image2 视觉资产记录

项目内视觉素材由内置 imagegen/image2 路径生成，最终文件位于：

- `src/assets/generated/imperial-desk.png`：桌面主背景与小安子肖像
- `src/assets/generated/app-icon-source.png`：应用图标源图
- `src-tauri/icons/`：由图标源图生成的桌面与移动规格

主视觉提示词要点：新中式皇帝书案、深色红木、宣纸、朱砂印、宫格窗影、右侧小太监肖像、中央大面积 UI 留白；禁止文字、水印、现代电子产品、过度装饰和伪汉字。

图标提示词要点：御案、展开的奏折、笔墨和克制朱印组成的原创方形徽记；禁止文字、龙冠、商标、3D 光泽与微小细节。

两项素材均使用内置生成模式完成，没有使用 CLI 或透明背景降级路径。
